## 🚀 Overview

TalentPulse AI is a full-stack AI-powered recruitment and talent intelligence platform designed to streamline the hiring process for both recruiters and candidates.

The platform provides:
- AI Resume Parsing & Analysis
- Real-time Recruiter-Candidate Workflow
- Job Posting & Application Tracking
- Candidate Profile Management
- Resume Intelligence Dashboards
- Smart Candidate-Job Matching
- Talent Analytics System

This project simulates a modern enterprise hiring ecosystem with separate recruiter and candidate portals connected through a real-time backend.

# 🔄 Workflow

## Candidate Flow

1. Candidate signs up
2. Completes profile onboarding
3. Uploads resume
4. AI parses resume automatically
5. Candidate explores live job feed
6. Applies to jobs
7. Tracks application status in real time

---

## Recruiter Flow

1. Recruiter creates company profile
2. Posts jobs
3. Views incoming applications
4. Reviews parsed resumes
5. Shortlists or rejects candidates
6. Tracks hiring analytics

# ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/talentpulse-ai.git

# Navigate into project
cd talentpulse-ai

# Install dependencies
npm install

# Run development server
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_GEMINI_API_KEY=your_gemini_key
```
